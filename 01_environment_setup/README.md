# Phase 01 — Environment Setup

**Module:** Module 1  
**Topics:** SAS interface, windows, LIBNAME, dataset basics, best practices

## What You Will Learn
- Navigate the SAS Enhanced Editor, Log, Output, and Explorer windows
- Understand the difference between temporary (WORK) and permanent SAS datasets
- Define a LIBNAME to point SAS to a folder of permanent datasets
- Apply basic SAS syntax rules and best practices

## Key Concepts
- `LIBNAME` statement — links a SAS library reference to a folder path
- `PROC CONTENTS` — inspect dataset variables, types, and lengths
- `PROC PRINT` — view dataset rows
- Temporary datasets live in `WORK`; permanent datasets need a libname
- SAS statements end with a semicolon `;`
