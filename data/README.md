# Data Directory

This folder contains all SAS datasets (`.sas7bdat`) used across phases 02–12, plus raw input files.

## Subdirectory

- `raw/` — original text and Excel files imported in Phase 02

## Dataset Index

| Dataset | Phase(s) Used | Description |
|---------|--------------|-------------|
| `phase1.sas7bdat` | 02, 03 | Phase 1 cohort — demographics, enrollment |
| `phase2.sas7bdat` | 02, 03 | Phase 2 follow-up — visit counts |
| `labs.sas7bdat` | 02, 03 | Laboratory results by subject |
| `cohort1516.sas7bdat` | 02 | 2015–16 enrollment cohort |
| `cohort1617.sas7bdat` | 02 | 2016–17 enrollment cohort |
| `survey.sas7bdat` | 02 | Survey response dataset |
| `mod3lab.sas7bdat` | 03 | Lab supplement for partial merge exercises |
| `mod4.sas7bdat` | 04 | Continuous variables for categorization |
| `atlgrades.sas7bdat` | 05, 12 | Student grade records |
| `mod6.sas7bdat` | 09 | Case-control dataset |
| `mock_data.sas7bdat` | 06 | Mixed-type data for function exercises |
| `mod8_a–d.sas7bdat` | 07 | Four datasets for array exercises |
| `pets.sas7bdat` | 07 | Variable naming exercise dataset |
| `patients.sas7bdat` | 08 | Patient records, multiple visits |
| `mod9.sas7bdat` | 08 | Multi-observation longitudinal dataset |
| `flbygroup.sas7bdat` | 08 | Flu surveillance, grouped |
| `patients_part2.sas7bdat` | 08 | Extended patient follow-up |
| `screener_data.sas7bdat` | 08 | Screening questionnaire responses |
| `tagsetex.sas7bdat` | 10 | ODS tagset example data |
| `aidsvu_2016_newdx_county.sas7bdat` | 12 | HIV new diagnoses by US county, 2016 |
| `final_clin.sas7bdat` | Final Exam | Clinical outcomes |
| `final_demo.sas7bdat` | Final Exam | Demographic data |

All datasets are synthetic or educational — no real patient or program records.
