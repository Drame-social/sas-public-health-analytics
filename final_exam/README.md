# Final Exam

Materials from the AEPI537D course final examination.

## Files
| File | Description |
|------|-------------|
| `AEPI537D_Final_Exam.docx` | Final exam questions (open-book, using SAS datasets provided) |

## Datasets Used (from `/data`)
`final_clin.sas7bdat` — clinical outcomes  
`final_demo.sas7bdat` — demographics

The exam tests synthesis of skills from all 12 phases: data import, merging, cleaning, transformation, descriptive analysis, and reporting.
